Consent Manager para Open Finance

Motor de consentimiento que permite a entidades financieras gestionar la autorización del usuario para compartir su información con terceros, cumpliendo con el Decreto 0368/2026 y estándares internacionales.

✅ Consent Manager

🔐ConsentimientoEl usuario decide qué datos comparte, con quién y por cuánto tiempo
⚡Ciclo de vidaCrear, autorizar, consultar, revocar y expirar consentimientos
🛡️SeguridadmTLS, FAPI 2.0, tokens vinculados a certificado, SCA obligatoria
📋AuditoriaLogs inmutables con hash encadenado, retención 5 años, exportable
🔔NotificacionEnterprise IAM
☁️Multi-NubeDesplegable en AWS, Azure o GCP con Terraform

🏗️ Ecosistema de Proveedores Integrados

RaidiamDirectory + PKI + DCR
AuthleteAuth Server FAPI 2.0
CurityIdentity Server
CloudentityAuthorization + Consent
Ping IdentityEnterprise IAM
Transmit SecurityBiometrics + Passkeys
