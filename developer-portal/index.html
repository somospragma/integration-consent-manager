🔑 Seguridad

FAPI 2.0 con proveedores certificados.

  • Auth Server: Authlete / Curity / Cloudentity / Ping
  • SCA: Transmit Security (biometrics, passkeys)
  • KYC: ConnectID (identity verification)
  • Directory: Raidiam (PKI + trust framework)

🔄 Flujo de Consentimiento

Paso a paso para obtener acceso a datos del usuario.

  • 1. POST /consents (crear)
  • 2. PAR + Authorize (SCA)
  • 3. POST /token (obtener access)
  • 4. GET /accounts (consumir)
